# Elobooks - Front-end

# Descrição:

Este projeto corresponde ao frontend do sistema Elobooks, responsável pela interface de interação do usuário com o sistema de Gestão de Biblioteca.
O frontend consome a API REST do backend para realizar operações como cadastro, consulta, empréstimo, devolução e recomendação de livros.

# Como rodar o projeto:

Este projeto foi desenvolvido em Angular com Node.js na versão 20.

Passos para rodar o front-end:

1. Você precisará do GIT para fazer o clone do projeto: 
git clone https://github.com/leandrolbnu/elobooks-frontend.git

2. Na pasta raíz do projeto, abra um cmd e digite: npm install

3. Em seguida, após ele instalar as dependências, rode o comando: npm start

4. Depois que estiver rodando, a aplicação estará acessível em:
http://localhost:4200


# Observações:

O backend deve estar em execução para o correto funcionamento do frontend. O link do repositório do backend é este:
https://github.com/leandrolbnu/elobooks

Certifique-se de que a API esteja acessível em http://localhost:8080
